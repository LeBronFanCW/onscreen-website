ONScreen Website
================

This website is completely static and works on Intel and Apple-silicon Macs.
It does not require Node.js, npm, a database, or an internet connection.

Quick start
-----------

1. Copy the entire ONScreen-Website folder to the other Mac.
2. Double-click index.html.

Optional local server
---------------------

If Python 3 is installed, open Terminal in this folder and run:

    python3 -m http.server 8080

Then visit:

    http://localhost:8080

The Download buttons point to downloads/ONScreen-1.3.dmg inside this folder.
Keep the folder structure intact when moving the website.
